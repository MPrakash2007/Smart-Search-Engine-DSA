# Smart Search Engine using DSA

A mini search engine built using Data Structures and Algorithms.

## Features

- Trie based autocomplete
- Spell correction using Edit Distance DP
- Document searching using Inverted Index
- Search ranking using Priority Queue
- Search history using Queue

## Data Structures Used

- Trie
- Hash Map
- Priority Queue
- Queue
- Dynamic Programming

## Technologies

- C++
- STL
- Data Structures and Algorithms

## How to Run

Compile:

g++ *.cpp -o SearchEngine

Run:

./SearchEngine
